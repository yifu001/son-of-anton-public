# Security Policy

## Supported Versions

We support the [latest released version](https://github.com/yifu001/son-of-anton/releases/latest), and the current development version (`master` branch).

## Reporting a Vulnerability

If you're concerned about a potential or proven security vulnerability in this software, please open a [security advisory](https://github.com/yifu001/son-of-anton/security/advisories/new) or create an issue with the `[SECURITY]` tag in the title.

For issues related to the original eDEX-UI codebase, you may also contact the original author at [`gabriel@saillard.dev`](mailto:gabriel@saillard.dev).
